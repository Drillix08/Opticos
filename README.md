# What is Opticos?

Opticos is an educational tool that uses interactive computer animations to explain fundamental concepts in highschool and college level math.

Right now we're currently working on covering the three main concepts in calculus 1, that being limits, derivatives, and integrals.

# Installation
Enter the directory you want to store Opticos the repository in and clone the repository using the following command
`git clone https://github.com/Drillix08/Opticos.git`
Then enter the Opticos directory:
`cd Opticos`

Opticos is made using the Godot game engine and thus you will have to install Godot to edit any of the code.
